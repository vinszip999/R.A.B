# R.A.B : reptiles & birds (파충류와 조류)


- ## 프로젝트 설명
    > 전세계의 다양한 파충류와 새들의 정보를 공유할 수 있는 웹 서비스입니다.
            
- ## 주요 기능
    1. 파충류와 조류에 대한 게시글 작성
    2. 게시한 글 목록 확인 기능
    3. 게시글에 댓글 및 좋아요 기능
    4. 좋아요 목록 확인 기능
        

- ## 프로젝트 실행
    1. python manage.py runserver
    2. configurations start
